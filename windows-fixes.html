Windows Fixes

Step-by-step guides for common Windows issues, with safe commands and restore points where relevant.

Last updated: June 5, 2026By Jordan Lee

Fix Windows Update Stuck at 0%

  1. Run Windows Update Troubleshooter: Settings → System → Troubleshoot → Other troubleshooters → Windows Update.
  2. Reset update services (Terminal as Admin): 
    net stop wuauserv
net stop cryptSvc & net stop bits & net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
ren C:\Windows\System32\catroot2 catroot2.old
net start wuauserv & net start cryptSvc
net start bits & net start msiserver
  3. Reboot, then check for updates again.

Tip: Ensure 10+ GB free disk space and pause VPNs during update.

Repair Corrupted System Files

  1. Open Terminal (Admin) and run: 
    sfc /scannow
  2. If issues persist, run DISM: 
    DISM /Online /Cleanup-Image /RestoreHealth
  3. Reboot and run sfc /scannow once more.

Note: Keep a recent restore point before major repairs.

Free Up Disk Space

  • Storage Sense: Settings → System → Storage → enable and run cleanup.
  • Disk Cleanup: Clean system files, remove old Windows updates.
  • Uninstall large apps: Settings → Apps (sort by size).

Speed Up Boot Time

  • Disable startup apps: Task Manager → Startup apps → Disable unnecessary.
  • Enable Fast Startup: Control Panel → Power Options.
  • Update GPU/chipset drivers; remove duplicate antivirus.
